              Assignment-03: Bootstrap Collapse with FAQ(Frequently Asked Questions)

              When one answer is expaned others answers will be collapsed. If we use an id at Accordion first line and data-bs-parent="#previously used id at first line" it will show only one accordion item at a time and others will automatically collapsed.

              Set up border is very important part of a web page. Using some Bootstrap class at first div we can set up any types of border easily.

              Hiding other answers is very important. If we can see all answers at a time it looks disgusting. Also we can save more space by hiding other answers.
